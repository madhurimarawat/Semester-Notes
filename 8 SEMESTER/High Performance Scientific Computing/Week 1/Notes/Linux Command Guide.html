Example: ls -l		 File Permissions The 10-character format: rwxr-xr--
Example: Owner (rwx), Group (r-x), Others (r--)
chmod u+x file.txt Adds execute permission for the user. chmod g-w file.txt Removes write permission from the group.
chmod a+r file.txt Gives read permission to all users. chmod 755 file.txt Sets permissions: owner (rwx), group (r-x), others (r-x).
chmod 644 file.txt Sets permissions: owner (rw-), group (r--), others (r--). chown user:group file.txt Changes the file owner and group.
ps Shows currently running processes.
Example: ps aux		 ps -e Lists all running processes.
ps -ef Lists all processes with full details. top Displays real-time system processes.
kill PID Terminates a process by PID. kill -9 PID Forces immediate termination of a process.
kill -SIGTERM PID Gracefully terminates a process. kill -SIGSTOP PID Pauses a process.
kill -SIGCONT PID Resumes a paused process. kill -SIGKILL PID Forces process termination without cleanup.
