Kernel The core of the OS, acting as a bridge between hardware and software. Process Management, Memory Management, Device Management, System Calls Linux Kernel, Windows NT Kernel, Monolithic & Microkernel Linux servers running cloud applications
Process Management Handles creation, scheduling, and termination of processes. Multitasking, CPU Scheduling, Process Synchronization, IPC Task Manager (Windows), htop (Linux), ps command Running multiple applications like a browser and a text editor simultaneously
Memory Management Manages system memory allocation and optimization. Memory Allocation, Paging, Segmentation, Protection Valgrind, RAMMap (Windows), Virtual Memory Systems Swapping unused applications to disk to free up RAM
File System Management Controls how data is stored and accessed on storage devices. File Creation, Access Control, Organization, Storage NTFS, EXT4, FAT32, File Explorer Saving and retrieving files from an external USB drive
Device Management Manages communication between OS and hardware peripherals. Device Drivers, I/O Management, Buffering, Spooling Device Manager (Windows), udev (Linux), Plug and Play Printing a document using a connected printer
User Interface (UI) Provides interaction via GUI or CLI. Graphical UI, Command-line Interface GNOME, KDE, Windows Explorer, Bash Using a web browser or executing commands in a terminal
Security & Access Control Protects the system from unauthorized access and threats. Authentication, Authorization, Encryption, Firewalls Firewalld, SELinux, Windows Defender, OpenSSL Logging into an online banking website securely
Networking Handles communication over LAN/WAN. Data Transmission, Network Security, Resource Sharing Wireshark, TCP/IP, DNS, VPN Connecting to Wi-Fi and browsing the internet
