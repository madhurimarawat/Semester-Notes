Measures of Central Tendency 📊

Properties 📜 Mean ➕ Median 📏 Mode 🔄 Quartiles 🧩
Definition 📖 Sum of all values divided by total number of values Middle value when data is arranged in ascending order Value that appears most frequently Divides data into four equal parts (Q1, Q2, Q3, Q4)
Formula 📝 Mean (μ) = (Σx) / N If odd: Median = middle value
If even: Median = (n/2th value + (n/2 + 1)th value) / 2 		Mode = Most frequent value in dataset Q1 = (1/4) * (N+1)th value
Q2 = Median
Q3 = (3/4) * (N+1)th value
Use Cases 🎯 Used for finding the overall trend, like average salary Used for skewed data like income distribution Used for categorical data like most popular product Used for detailed distribution analysis in datasets
Computation Complexity 🖥️ Easy: O(n) Moderate: O(n log n) (sorting required) Easy: O(n) Moderate: O(n log n) (sorting required)
Effect of Outliers ⚠️ Highly affected Less affected Not affected Not affected
Best Used For ✅ Symmetric distributions Skewed distributions Nominal and categorical data Detailed breakdown of dataset distribution
Real-Life Examples 🌟 Average exam scores, temperature Median income, median house price Most popular mobile phone brand Salary quartiles, age distribution
